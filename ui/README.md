# Todo App in React+Redux


## Download Project

Download the project through Git

`git clone https://github.com/nomanHasan/todoapp-react`

`cd todoapp-react`

## Development Server
First Run `yarn` or `npm install`

Then `npm start`


## Libraries used

* Axios
* Semantic-UI-React
* Redux
* Redux-Thunk
* Redux-logger
* React-Router
* React-Datepicker
* Moment
